# airsstack

Execution engine for the airsstack development methodology, as a Claude Code plugin.

Ships (after later objectives): four agents — `coder`, `reviewer`, `verifier`, `explorer` — an
`orchestrate` driver skill, `process-guidelines`, project-local memory skills, a `concise` output
mode, and a terse output style.

## Install

```
/plugin marketplace add rstlix0x0/airsstack
/plugin install airsstack@airsstack
```

Skills and agents are namespaced `airsstack:<name>`.
