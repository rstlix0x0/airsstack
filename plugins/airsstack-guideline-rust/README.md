# airsstack-guideline-rust

Rust engineering guidelines and Definition-of-Done for Claude Code, delivered as a single
lazily-loaded skill (`airsstack-guideline-rust:rust-guidelines`).

The airsstack execution agents invoke this skill to obtain the Rust DoD and rules when they edit Rust
code; it is also useful on its own.

## Install

```
/plugin marketplace add rstlix0x0/airsstack
/plugin install airsstack-guideline-rust@airsstack
```
